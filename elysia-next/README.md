# Elysia Next

This project uses ElysiaJS with the new Next.js 16. In this project, I would use ElysiaJS API instead of Next.js default APIs.

## Why ElysiaJS?

ElysiaJS offers a more powerful and flexible API compared to Next.js default APIs. With ElysiaJS, I can create a more robust and scalable application.

## Features

- Using ElysiaJS API instead of Next.js default APIs
- Fully compatible with Next.js 16
- Easy to use and integrate with existing Next.js projects
